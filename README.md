# pcb-usb-audio-viewer

Interactive PCB viewer for the USB Audio Board (18.1 × 35.4 mm).

**Live:** https://gaorong77.github.io/pcb-usb-audio-viewer/

## Features

- Board outline (line + arc segments)
- Route keepin boundary
- 58 components with bbox display
- Clearance layer (0.075 mm buffer)
- Feasible routing area (keepin − clearance = 348.6 mm²)
- 165 pins color-coded by net
- Net filter, hover tooltip, pan/zoom
